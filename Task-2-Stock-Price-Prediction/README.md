# Task 2: Short-Term Stock Price Prediction

## Objective
To predict the next day's closing stock price using historical stock market data.

## Dataset
- Source: Yahoo Finance
- Library used: yfinance
- Stock: Apple Inc. (AAPL)

## Features Used
- Open
- High
- Low
- Volume

## Model Applied
- Linear Regression

## Evaluation
- Compared actual vs predicted closing prices
- Visualized results using plots

## Key Results
- The model shows reasonable performance for short-term prediction.
- Stock prices are volatile; predictions depend on market conditions.

## Skills Demonstrated
- Time series data handling
- Regression modeling
- Data fetching using APIs (yfinance)
- Data visualization
