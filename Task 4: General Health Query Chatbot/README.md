# General Health Chatbot (Hugging Face API)

A simple **health-related chatbot** built using Hugging Face Inference API.  
It uses prompt engineering and safety filtering to provide **general health information**.

---

## Features

- Friendly and clear responses for general health questions
- Safety filter for emergency situations
- Prompt engineering to guide model responses
- Cloud-based model (no heavy downloads)
- Works directly in Google Colab or local Python

---

## Requirements

- Python 3.8+
- `requests` library
- Hugging Face API token

Install required library:

```bash
pip install requests
